# visit-with-us-wellness

This repository implements an end-to-end MLOps pipeline to predict
customer purchases of a Wellness Tourism Package.

### Tech Stack
- Python
- Scikit-learn
- GitHub Actions (CI/CD)
- Hugging Face Hub & Spaces
- Streamlit
